# Taskito Docker

Taskito Docker es una ToDo app simple hecha con React + Vite y lista para desplegar con Docker Compose.

## Requisitos

- Docker
- Docker Compose

## Ejecutar con Docker Compose

```bash
docker compose up --build
```

Abre la app en:

```text
http://localhost:8080
```

## Detener

```bash
docker compose down
```

## Ejecutar en modo desarrollo sin Docker

```bash
npm install
npm run dev
```

Abre:

```text
http://localhost:5173
```
